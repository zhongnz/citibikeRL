# Proposal Folder

Store all proposal artifacts here.

## Suggested files
- `proposal_outline.md`
- `proposal_draft_v1.md`
- `proposal_final.pdf`
- `figures/` (optional subfolder for proposal-only images)

## Proposal checklist
- [ ] Problem statement clearly defined
- [ ] MDP state/action/reward summarized
- [ ] Baseline policy defined
- [ ] Dataset source and scope identified
- [ ] Evaluation metrics listed
- [ ] Risks and mitigation documented
