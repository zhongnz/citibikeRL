# Report Folder

Store all report materials and versions here.

## Suggested files
- `report_outline.md`
- `report_draft_v1.md`
- `report_draft_v2.md`
- `report_final.pdf`

## Recommended section order
1. Abstract
2. Introduction
3. Problem setup and MDP design
4. Dataset and preprocessing assumptions
5. Methods (Q-learning; optional DQN extension)
6. Baseline and evaluation metrics
7. Results
8. Discussion and limitations
9. Conclusion
