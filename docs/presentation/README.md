# Presentation Folder

Store slides and talk materials.

## Suggested files
- `slide_outline.md`
- `deck_v1.pptx` / `deck_final.pptx`
- `speaker_notes.md`
- `backup_slides.md`

## Slide flow (recommended)
1. Problem and motivation
2. Environment/MDP definition
3. Data and assumptions
4. Method + baseline
5. Results and interpretation
6. Limitations
7. Next steps
